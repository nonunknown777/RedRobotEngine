\doxysubsection{Detailed Description}
Decimal unit prefices in the metric system for denoting a multiple, or a fraction of, a unit. 

\begin{DoxyNote}{Note}
{\ttfamily std\+::ratio} doesn\textquotesingle{}t necessarily support larger sizes like Yotta or bigger so floating-\/point literal notation is used instead.

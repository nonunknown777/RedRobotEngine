\doxysubsection{Detailed Description}
Units file. 

This file establishes quantity aliases, unit constants, and associated code for the expression of physical quantities using recognizably named units of those quantities. Quantities, as used herein, are types associated with physical quantities like time and length. Conceptually a given quantity is only expressable in units that relate to that quantity. For every quantity defined herein, there\textquotesingle{}s typically at least one conceptually typed unit asscociated with it.

\begin{DoxyNote}{Note}
In the simplest way that the Play\+Rho library can be configured, there\textquotesingle{}s no compiler enforcement on the usage of the units for their associated quantities beyond the usage of the Real type. 
\end{DoxyNote}

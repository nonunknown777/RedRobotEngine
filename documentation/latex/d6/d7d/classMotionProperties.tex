\doxysubsection{Detailed Description}
The \doxylink{classBody}{Body} class only keeps track of state for static bodies, the \doxylink{classMotionProperties}{Motion\+Properties} class keeps the additional state needed for a moving \doxylink{classBody}{Body}. It has a 1-\/on-\/1 relationship with the body.

\doxysubsection{Detailed Description}
Exceptions defined and used by the Play\+Rho library. 

The Play\+Rho library defines its own exception classes to recognize errors. These classes are all sub-\/classed from sub-\/classes of the C++ Standard Library std\+::exception class.

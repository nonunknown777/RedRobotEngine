\doxysubsection{Detailed Description}
Class that can construct shapes and that is serializable using the \doxylink{classObjectStream}{Object\+Stream} system. Can be used to store shape data in \textquotesingle{}uncooked\textquotesingle{} form (i.\+e. in a form that is still human readable and authorable). Once the shape has been created using the \doxylink{classShapeSettings_a21e53b5758542a5527c7941d61d838fa}{Create()} function, the data will be moved into the \doxylink{classShape}{Shape} class in a form that is optimized for collision detection. After this, the \doxylink{classShapeSettings}{Shape\+Settings} object is no longer needed and can be destroyed. Each shape class has a derived class of the \doxylink{classShapeSettings}{Shape\+Settings} object to store shape specific data.

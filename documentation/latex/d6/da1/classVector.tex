Calculates the matrix product of the two given vector of vectors (matrices). 

Multiplies an A-\/by-\/B vector of vectors by a B-\/by-\/C vector of vectors returning an A-\/by-\/C vector of vectors. \begin{DoxyNote}{Note}
From Wikipedia\+: \begin{quote}
Multiplication of two matrices is defined if and only if the number of columns of the left matrix is the same as the number of rows of the right matrix. \end{quote}


\doxylink{classMatrix}{Matrix} multiplication is not commutative. 

Algorithmically speaking, this implementation is called the "{}naive"{} algorithm. For small matrices, like 3-\/by-\/3 or smaller matrices, its complexity shouldn\textquotesingle{}t be an issue. The matrix dimensions are compile time constants anyway which can help compilers automatically identify loop unrolling and hardware level parallelism opportunities. 
\end{DoxyNote}

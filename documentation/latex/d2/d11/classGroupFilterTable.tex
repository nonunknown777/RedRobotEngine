\doxysubsection{Detailed Description}
Implementation of \doxylink{classGroupFilter}{Group\+Filter} that stores a bit table with one bit per sub shape ID pair to determine if they collide or not

The collision rules\+:
\begin{DoxyItemize}
\item If one of the objects is in the c\+Invalid\+Group the objects will collide.
\item If the objects are in different groups they will collide.
\item If they\textquotesingle{}re in the same group but their collision filter is different they will not collide.
\item If they\textquotesingle{}re in the same group and their collision filters match, we\textquotesingle{}ll use the Sub\+Group\+ID and the table below.
\end{DoxyItemize}

For N = 6 sub groups the table will look like\+: \begin{DoxyVerb}            sub group 1 --->
sub group 2 x.....
     |      ox....
     |      oox...
     V      ooox..
            oooox.
            ooooox
\end{DoxyVerb}

\begin{DoxyItemize}
\item \textquotesingle{}x\textquotesingle{} means sub group 1 == sub group 2 and we define this to never collide.
\item \textquotesingle{}o\textquotesingle{} is a bit that we have to store that defines if the sub groups collide or not.
\item \textquotesingle{}.\textquotesingle{} is a bit we don\textquotesingle{}t need to store because the table is symmetric, we take care that group 2 \texorpdfstring{$>$}{>} group 1 by swapping sub group 1 and sub group 2 if needed.
\end{DoxyItemize}

The total number of bits we need to store is (N \texorpdfstring{$\ast$}{*} (N -\/ 1)) / 2

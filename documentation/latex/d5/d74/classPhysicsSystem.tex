\doxysubsection{Detailed Description}
The main class for the physics system. It contains all rigid bodies and simulates them.

The main simulation is performed by the \doxylink{classPhysicsSystem_a4cdc76b2486dfaecfc9319d040e5e5f3}{Update()} call on multiple threads (if the \doxylink{classJobSystem}{Job\+System} is configured to use them). Please refer to the general architecture overview in the Docs folder for more information.

Check if 2 bodies were in contact during the last simulation step. Since contacts are only detected between active bodies, so at least one of the bodies must be active in order for this function to work. It queries the state at the time of the last \doxylink{classPhysicsSystem_a4cdc76b2486dfaecfc9319d040e5e5f3}{Physics\+System\+::\+Update} and will return true if the bodies were in contact, even if one of the bodies was moved / removed afterwards. This function can be called from any thread when the \doxylink{classPhysicsSystem_a4cdc76b2486dfaecfc9319d040e5e5f3}{Physics\+System\+::\+Update} is not running. During \doxylink{classPhysicsSystem_a4cdc76b2486dfaecfc9319d040e5e5f3}{Physics\+System\+::\+Update} this function is only valid during contact callbacks\+:
\begin{DoxyItemize}
\item During the \doxylink{classContactListener_aeb1d7908c8556ce24448ac572d7ad944}{Contact\+Listener\+::\+On\+Contact\+Added} callback this function can be used to determine if a different contact pair between the bodies was active in the previous simulation step (function returns true) or if this is the first step that the bodies are touching (function returns false).
\item During the \doxylink{classContactListener_aef8f0cac1d560622743420afcd1ce17a}{Contact\+Listener\+::\+On\+Contact\+Removed} callback this function can be used to determine if this is the last contact pair between the bodies (function returns false) or if there are other contacts still present (function returns true).

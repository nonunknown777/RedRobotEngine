\doxysubsubsection{\texorpdfstring{IsActive()}{IsActive()}}
{\footnotesize\ttfamily bool Ragdoll\+::\+Is\+Active (\begin{DoxyParamCaption}\item[{bool}]{in\+Lock\+Bodies = {\ttfamily true} }\end{DoxyParamCaption}) const}

Check if one or more of the bodies in the ragdoll are active. Note that this involves locking the bodies (if in\+Lock\+Bodies is true) and looping over them. An alternative and possibly faster way could be to install a \doxylink{classBodyActivationListener}{Body\+Activation\+Listener} and count the number of active bodies of a ragdoll as they\textquotesingle{}re activated / deactivated (basically check if the body that activates / deactivates is in \doxylink{classRagdoll_a98603396696b276256ffe74283a26fca}{Get\+Body\+IDs()} and increment / decrement a counter).

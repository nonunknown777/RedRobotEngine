\item \doxylink{classSphereShape}{Sphere\+Shape}\+: Scale must be uniform (signs of scale are ignored).
\item \doxylink{classBoxShape}{Box\+Shape}\+: Any scale supported (signs of scale are ignored).
\item \doxylink{classTriangleShape}{Triangle\+Shape}\+: Any scale supported when convex radius is zero, otherwise only uniform scale supported.
\item \doxylink{classCapsuleShape}{Capsule\+Shape}\+: Scale must be uniform (signs of scale are ignored).
\item \doxylink{classTaperedCapsuleShape}{Tapered\+Capsule\+Shape}\+: Scale must be uniform (sign of Y scale can be used to flip the capsule).
\item \doxylink{classCylinderShape}{Cylinder\+Shape}\+: Scale must be uniform in XZ plane, Y can scale independently (signs of scale are ignored).
\item \doxylink{classRotatedTranslatedShape}{Rotated\+Translated\+Shape}\+: Scale must not cause shear in the child shape.
\item \doxylink{classCompoundShape}{Compound\+Shape}\+: Scale must not cause shear in any of the child shapes.

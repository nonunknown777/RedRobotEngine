\doxysubsection{Detailed Description}
\doxylink{classObjectStreamIn}{Object\+Stream\+In} contains all logic for reading an object from disk. It is the base class for the text and binary input streams (\doxylink{classObjectStreamTextIn}{Object\+Stream\+Text\+In} and \doxylink{classObjectStreamBinaryIn}{Object\+Stream\+Binary\+In}).

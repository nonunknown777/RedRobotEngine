\doxysubsection{Detailed Description}
\doxylink{classObjectStreamOut}{Object\+Stream\+Out} contains all logic for writing an object to disk. It is the base class for the text and binary output streams (\doxylink{classObjectStreamTextOut}{Object\+Stream\+Text\+Out} and \doxylink{classObjectStreamBinaryOut}{Object\+Stream\+Binary\+Out}).

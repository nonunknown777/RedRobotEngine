\doxysubsection{Detailed Description}
Light weight runtime type information system. This way we don\textquotesingle{}t need to turn on the default \doxylink{classRTTI}{RTTI} system of the compiler (introducing a possible overhead for every class)

Notes\+:
\begin{DoxyItemize}
\item An extra virtual member function is added. This adds 8 bytes to the size of an instance of the class (unless you are already using virtual functions).
\end{DoxyItemize}

To use \doxylink{classRTTI}{RTTI} on a specific class use\+:

Header file\+: \begin{DoxyVerb}class Foo
{
    JPH_DECLARE_RTTI_VIRTUAL_BASE(Foo)
}

class Bar : public Foo
{
    JPH_DECLARE_RTTI_VIRTUAL(Bar)
};
\end{DoxyVerb}
 Implementation file\+: \begin{DoxyVerb}JPH_IMPLEMENT_RTTI_VIRTUAL_BASE(Foo)
{
}

JPH_IMPLEMENT_RTTI_VIRTUAL(Bar)
{
    JPH_ADD_BASE_CLASS(Bar, Foo) // Multiple inheritance is allowed, just do JPH_ADD_BASE_CLASS for every base class
}
\end{DoxyVerb}
 For abstract classes use\+:

Header file\+: \begin{DoxyVerb}class Foo
{
    JPH_DECLARE_RTTI_ABSTRACT_BASE(Foo)

public:
    virtual void AbstractFunction() = 0;
}

class Bar : public Foo
{
    JPH_DECLARE_RTTI_VIRTUAL(Bar)

public:
    virtual void AbstractFunction() { } // Function is now implemented so this class is no longer abstract
};
\end{DoxyVerb}
 Implementation file\+: \begin{DoxyVerb}JPH_IMPLEMENT_RTTI_ABSTRACT_BASE(Foo)
{
}

JPH_IMPLEMENT_RTTI_VIRTUAL(Bar)
{
    JPH_ADD_BASE_CLASS(Bar, Foo)
}
\end{DoxyVerb}
 Example of usage in a program\+: \begin{DoxyVerb}Foo *foo_ptr = new Foo;
Foo *bar_ptr = new Bar;

IsType(foo_ptr, RTTI(Bar)) returns false
IsType(bar_ptr, RTTI(Bar)) returns true

IsKindOf(foo_ptr, RTTI(Bar)) returns false
IsKindOf(bar_ptr, RTTI(Foo)) returns true
IsKindOf(bar_ptr, RTTI(Bar)) returns true

StaticCast<Bar>(foo_ptr) asserts and returns foo_ptr casted to pBar
StaticCast<Bar>(bar_ptr) returns bar_ptr casted to pBar

DynamicCast<Bar>(foo_ptr) returns nullptr
DynamicCast<Bar>(bar_ptr) returns bar_ptr casted to pBar
\end{DoxyVerb}
 Other feature of Dynamic\+Cast\+: \begin{DoxyVerb}class A { int data[5]; };
class B { int data[7]; };
class C : public A, public B { int data[9]; };

C *c = new C;
A *a = c;
\end{DoxyVerb}
 Note that\+: \begin{DoxyVerb}B *b = (B *)a;
\end{DoxyVerb}
 generates an invalid pointer, \begin{DoxyVerb}B *b = StaticCast<B>(a);
\end{DoxyVerb}
 doesn\textquotesingle{}t compile, and \begin{DoxyVerb}B *b = DynamicCast<B>(a);
\end{DoxyVerb}
 does the correct cast

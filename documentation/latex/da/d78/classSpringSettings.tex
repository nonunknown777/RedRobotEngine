\doxysubsubsection{\texorpdfstring{mDamping}{mDamping}}
{\footnotesize\ttfamily float Spring\+Settings\+::m\+Damping = 0.\+0f}

When m\+Spring\+Mode = ESpring\+Mode\+::\+Frequency\+And\+Damping m\+Damping is the damping ratio (0 = no damping, 1 = critical damping). When m\+Spring\+Mode = ESpring\+Mode\+::\+Stiffness\+And\+Damping m\+Damping is the damping (c) in the spring equation F = -\/k \texorpdfstring{$\ast$}{*} x -\/ c \texorpdfstring{$\ast$}{*} v for a linear or T = -\/k \texorpdfstring{$\ast$}{*} theta -\/ c \texorpdfstring{$\ast$}{*} w for an angular spring. Note that if you set m\+Damping = 0, you will not get an infinite oscillation. Because we integrate physics using an explicit Euler scheme, there is always energy loss. This is done to keep the simulation from exploding, because with a damping of 0 and even the slightest rounding error, the oscillation could become bigger and bigger until the simluation explodes. \Hypertarget{classSpringSettings_acea8e3f5767edbfaffa01417733b5dc0}\label{classSpringSettings_acea8e3f5767edbfaffa01417733b5dc0}

\doxysubsubsection{\texorpdfstring{mStiffness}{mStiffness}}
{\footnotesize\ttfamily float Spring\+Settings\+::m\+Stiffness}

Valid when m\+Spring\+Mode = ESpring\+Mode\+::\+Stiffness\+And\+Damping. If m\+Stiffness \texorpdfstring{$>$}{>} 0 the constraint will be soft and m\+Stiffness specifies the stiffness (k) in the spring equation F = -\/k \texorpdfstring{$\ast$}{*} x -\/ c \texorpdfstring{$\ast$}{*} v for a linear or T = -\/k \texorpdfstring{$\ast$}{*} theta -\/ c \texorpdfstring{$\ast$}{*} w for an angular spring. If m\+Stiffness \texorpdfstring{$<$}{<}= 0, m\+Damping is ignored and the constraint will have hard limits (as hard as the time step / the number of velocity / position solver steps allows).

Note that stiffness values are large numbers. To calculate a ballpark value for the needed stiffness you can use\+: force = stiffness \texorpdfstring{$\ast$}{*} delta\+\_\+spring\+\_\+length = mass \texorpdfstring{$\ast$}{*} gravity \texorpdfstring{$<$}{<}=\texorpdfstring{$>$}{>} stiffness = mass \texorpdfstring{$\ast$}{*} gravity / delta\+\_\+spring\+\_\+length. So if your object weighs 1500 kg and the spring compresses by 2 meters, you need a stiffness in the order of 1500 \texorpdfstring{$\ast$}{*} 9.\+81 / 2 \texorpdfstring{$\sim$}{\string~} 7500 N/m.

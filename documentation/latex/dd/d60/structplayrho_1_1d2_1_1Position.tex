\doxysubsection{Detailed Description}
2-\/D positional data structure. 

A 2-\/element length and angle pair suitable for representing a linear and angular position in 2-\/D. \begin{DoxyNote}{Note}
This structure is likely to be 12-\/bytes large (at least on 64-\/bit platforms). 
\end{DoxyNote}

\doxysubsection{Detailed Description}
Implementation of the Barrier class for a \doxylink{classJobSystem}{Job\+System}

This class can be used to make it easier to create a new \doxylink{classJobSystem}{Job\+System} implementation that integrates with your own job system. It will implement all functionality relating to barriers, so the only functions that are left to be implemented are\+:

\item \doxylink{classJobSystem_a0766c8acd5ed9ef0e6baf1c2b8f3c38b}{Job\+System\+::\+Create\+Job}
\item \doxylink{classJobSystem_aa74329d956a01e3f2f219ea22bdc5097}{Job\+System\+::\+Free\+Job}
\item \doxylink{classJobSystem_ae6bae9c5725c2009c8e9a2d79f60cb40}{Job\+System\+::\+Queue\+Job}/\+Queue\+Jobs
\end{DoxyItemize}

See instructions in \doxylink{classJobSystem}{Job\+System} for more information on how to implement these.

\doxysubsection{Detailed Description}
Types for physical quantities. 

These are the type aliases for physical quantities like time and length that are used by the Play\+Rho library. Conceptually a given quantity is only expressable in the units that are defined for that quantity. \begin{DoxySeeAlso}{See also}

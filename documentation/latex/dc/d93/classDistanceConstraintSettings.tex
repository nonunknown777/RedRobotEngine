\doxylink{classBody}{Body} 1 constraint reference frame (space determined by m\+Space). \doxylink{classConstraint}{Constraint} will keep m\+Point1 (a point on body 1) and m\+Point2 (a point on body 2) at the same distance. Note that this constraint can be used as a cheap \doxylink{classPointConstraint}{Point\+Constraint} by setting m\+Point1 = m\+Point2 (but this removes only 1 degree of freedom instead of 3).

\doxysubsection{Detailed Description}
Units for expressing physical quantities like time and length. 

These are the unit definitions for expressing physical quantities like time and length. Conceptually a given unit is only usable with the quantities that are made up of the dimensions which the unit is associated with. \begin{DoxySeeAlso}{See also}

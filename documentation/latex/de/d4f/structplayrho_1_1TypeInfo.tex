The name of the templated type. 

\begin{DoxyNote}{Note}
This is also a static member providing a unique ID, via its address, for the type T without resorting to using C++ run-\/time type information (\doxylink{classRTTI}{RTTI}). 

Setting this to a null-\/terminated byte string that\textquotesingle{}s unique to at least the template\textquotesingle{}s type {\ttfamily T} prevents issue \#370. Credit for this technique goes to Li Jin (github user pigpigyyy). 
\end{DoxyNote}

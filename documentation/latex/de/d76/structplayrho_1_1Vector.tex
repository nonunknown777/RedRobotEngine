\doxylink{structplayrho_1_1Vector}{Vector}. 

This is a {\ttfamily constexpr} and constructor enhanced {\ttfamily std\+::array}-\/like template class for types supporting the +, -\/, \texorpdfstring{$\ast$}{*}, / arithmetic operators ("{}arithmetic types"{} as defined by the {\ttfamily \doxylink{structplayrho_1_1IsArithmetic}{Is\+Arithmetic}} type trait) that itself comes with non-\/member arithmetic operator support making \doxylink{structplayrho_1_1Vector}{Vector} instances arithmetic types as well. \begin{DoxyNote}{Note}
This type is trivially default constructible -\/ i.\+e. default construction performs no actions (no initialization). 
\end{DoxyNote}

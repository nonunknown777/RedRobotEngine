\doxysubsection{Detailed Description}
Pulley constraint settings, used to create a pulley constraint. A pulley connects two bodies via two fixed world points to each other similar to a distance constraint. We define Length1 = \texorpdfstring{$\vert$}{|}\+Body\+Point1 -\/ Fixed\+Point1\texorpdfstring{$\vert$}{|} where Body1 is a point on body 1 in world space and Fixed\+Point1 a fixed point in world space Length2 = \texorpdfstring{$\vert$}{|}\+Body\+Point2 -\/ Fixed\+Point2\texorpdfstring{$\vert$}{|} The constraint keeps the two line segments constrained so that Min\+Distance \texorpdfstring{$<$}{<}= Length1 + Ratio \texorpdfstring{$\ast$}{*} Length2 \texorpdfstring{$<$}{<}= Max\+Distance

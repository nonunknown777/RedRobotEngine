Gets the template type parameter\textquotesingle{}s type name as a non-\/empty unique string. 

\begin{DoxyNote}{Note}
This string is the demangled name on supporting compilers, otherwise the mangled name is used and C++ run-\/time type information (\doxylink{classRTTI}{RTTI}) needs to be available. 

This code relies on the compiler being clang or GCC compatible supporting the {\ttfamily {\bfseries{clang}}} or {\ttfamily {\bfseries{GNUC}}} preprocessor macro and the {\ttfamily {\bfseries{PRETTY\+\_\+\+FUNCTION}}} identifier, or the compiler supporting the Microsoft Visual C++ style {\ttfamily {\bfseries{FUNCSIG}}} macro. C++20\textquotesingle{}s support for {\ttfamily std\+::source\+\_\+location\+::function\+\_\+name} could help make this function more portable, but only slightly since the returned string\textquotesingle{}s format is still implementation defined. 
\end{DoxyNote}

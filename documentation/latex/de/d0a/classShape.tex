\doxysubsubsection*{Binary serialization of the shape. Note that this saves the \textquotesingle{}cooked\textquotesingle{} shape in a format which will not be backwards compatible for newer library versions.}

The user is expected to call Save\+Binary\+State followed by Save\+Material\+State and Save\+Sub\+Shape\+State. The stream should be stored as is and the material and shape list should be saved using the applications own serialization system (e.\+g. by assigning an ID to each pointer). When restoring data, call s\+Restore\+From\+Binary\+State to get the shape and then call Restore\+Material\+State and Restore\+Sub\+Shape\+State to restore the pointers to the external objects.

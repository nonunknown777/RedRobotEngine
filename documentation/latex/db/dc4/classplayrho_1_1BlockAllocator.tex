\doxysubsection{Detailed Description}
Block allocator.

This is a small object allocator used for allocating small objects that persist for more than one time step. \begin{DoxyNote}{Note}
This data structure is 136-\/bytes large (on at least one 64-\/bit platform). 
\end{DoxyNote}

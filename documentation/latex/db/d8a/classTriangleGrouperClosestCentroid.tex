\doxysubsection{Detailed Description}
A class that groups triangles in batches of N. Starts with centroid with lowest X coordinate and finds N closest centroids, this repeats until all groups have been found. Time complexity\+: O(\+N\texorpdfstring{$^\wedge$}{\string^}2)

\doxysubsection{Detailed Description}
Interface to the broadphase that can perform collision queries. These queries will only test the bounding box of the body to quickly determine a potential set of colliding bodies. The shapes of the bodies are not tested, if you want this then you should use the \doxylink{classNarrowPhaseQuery}{Narrow\+Phase\+Query} interface.

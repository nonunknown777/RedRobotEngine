\doxysubsection{Detailed Description}
This class calculates the intersection between a fluid surface and a polyhedron and returns the submerged volume and its center of buoyancy Construct this class and then one by one add all faces of the polyhedron using the Add\+Face function. After all faces have been added the result can be gotten through Get\+Result.

\doxysubsection{Detailed Description}
Two objects collide with each other if\+:
\begin{DoxyItemize}
\item Both don\textquotesingle{}t have a group filter
\item The first group filter says that the objects can collide
\item Or if there\textquotesingle{}s no filter for the first object, the second group filter says the objects can collide 
\end{DoxyItemize}

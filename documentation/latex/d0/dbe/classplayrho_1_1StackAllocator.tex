\doxysubsection{Detailed Description}
Stack allocator.

This is a stack allocator used for fast per step allocations. You must nest allocate/free pairs. The code will assert if you try to interleave multiple allocate/free pairs. \begin{DoxyNote}{Note}
This class satisfies the C++11 {\ttfamily std\+::unique\+\_\+ptr()} Deleter requirement. 

This data structure is 64-\/bytes large (on at least one 64-\/bit platform). 
\end{DoxyNote}

\doxylink{classplayrho_1_1Interval}{Interval} template type. 

This type encapsulates an interval as a min-\/max value range relationship. \begin{DoxyInvariant}{Invariant}
The min and max values can only be the result of {\ttfamily std\+::minmax(a, b)} or the special values of the "{}highest"{} and "{}lowest"{} values supported by the type for this class respectively indicating the "{}unset"{} value. 
\end{DoxyInvariant}

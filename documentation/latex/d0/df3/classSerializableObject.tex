\doxysubsection{Detailed Description}
Classes must be derived from \doxylink{classSerializableObject}{Serializable\+Object} if you want to be able to save pointers or reference counting pointers to objects of this or derived classes. The type will automatically be determined during serialization and upon deserialization it will be restored correctly.
